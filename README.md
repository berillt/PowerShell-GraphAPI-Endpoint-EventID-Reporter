Kurumsal Endpoint Olay Raporlayıcı
Bu proje, kurumsal ağdaki cihazlardan (Intune üzerinden) gelen karmaşık olay loglarını toplar, kritik hataları analiz eder ve Microsoft Graph API kullanarak yönetici ekibine temiz, anlamlandırılmış bir özet rapor sunar.

Neden Bu Proje?

Sıfır Maliyet: Azure Sentinel veya Log Analytics gibi ücretli servisler yerine mevcut Microsoft 365 lisanslarını kullanır.

Gürültü Kirliliği Yok: Binlerce satır yerine sadece kritik ID'leri (Disk hataları, sürücü sorunları vb.) gruplayarak raporlar.

Tam Otomatik: Manuel hiçbir müdahale gerektirmez.



Enterprise Endpoint Event Reporter
Description: This project collects complex event logs from corporate devices (via Intune), analyzes critical errors, and delivers a clean, meaningful summary report to the IT administration using Microsoft Graph API.

Why This Project?

Zero Cost: Utilizes existing Microsoft 365 licenses instead of paid services like Azure Sentinel or Log Analytics.

Noise Reduction: Instead of thousands of lines, it groups and reports only critical IDs (Disk errors, driver issues, etc.).

Fully Automated: Requires no manual intervention.
